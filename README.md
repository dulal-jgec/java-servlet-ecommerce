# Java Servlet Shopping Cart

## About

This is a simple **E-commerce web application** built using **Java Servlet, JSP, JDBC and MySQL**.
Users can browse products, add them to the cart, place orders and manage their orders.

## Features

* Product listing
* Add to cart
* Remove from cart
* Increase / decrease quantity
* Buy now
* Checkout cart
* Order history
* Cancel order
* User login / logout

## Tech Stack

* Java Servlet
* JSP
* JDBC
* MySQL
* Bootstrap
* Apache Tomcat

## How to Run

1. Clone the repository
2. Import project into Eclipse
3. Configure Apache Tomcat
4. Setup MySQL database
5. Run the project
